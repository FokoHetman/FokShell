# FokShell
To put it simply, it is a project targetting to create a haskell-configurable shell with practical features and extensions.

## Motive
I just wanted to make my own shell. That's it. I didn't like zsh or fish, and bash was my go-to since forever.
I started developing it around the time I got heavily invested into haskell, and xmonad's configuration design was a huge inspiration for my language choice.

## Why choose FokShell over anything else?
Right now, there is zero reason to, and you're a psychopath if you actually unironically use fokshell.
In the future, I hope this will change however.
